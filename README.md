# Install-RDP
di Tutorial kali ini, kita akan membahas cara instal RDP 

## 1. Cek Bot Masih Berfungsi Atau Tidak
  https://t.me/instalrdpbot   
  `Cek Tipe Ubuntu Apa Yang Support Untuk Instalasi kita`   
  kalau masih berfungsi lanjut tahap selanjutnya   
  
## 2. Deploy VPS
  - `For Bot Reply` biasanya menggunakan spek VPS 8 RAM / 4 CPU
  - `For Bot Reply` Gunakan regional Singapura
  - karena nanti menggunakan instalasi pihak ketiga gunakan password sederhana ```11111Aaaaa```, Setelah RDP terinstal ganti lagi passwordnya

### Langkah-langkah mengganti password setalah RDP siap
  1. ```bash
     passwd
     ```
  2. masukan password baru

## 3. Instalasi RDP di Bot
  - `For Bot Reply` Gunakan windows 2019
  - Catat informasi yang diperlukan yang dihasilkan dari BOT
